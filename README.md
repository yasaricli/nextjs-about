# nextjs-about
The app you have about using nextjs.

### Installation
Clone repo:
```sh
git clone https://github.com/yasaricli/nextjs-about.git
cd nextjs-about
```

Install the dependencies:
```sh
npm install
```

### Development Workflow
Start a live-reload development server:
```sh
npm run dev
```

Generate a production build:
```sh
npm run build
```
